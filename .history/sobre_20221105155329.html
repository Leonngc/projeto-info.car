Sobre a Info .car

Quem somos?

A Info .car é um projeto de site feito em HTML, CSS e Javascript (este está na página Mais ).
Somos uma página que hospeda notícias automotivas das mais diferentes páginas. Esse projeto é somente de Portfólio, sendo sem monetização ou quaisquer ganho com conteúdo de outras páginas e empresas.
Para você amante de carros, motos e velocidade, essa é sua página predileta a partir de agora.
